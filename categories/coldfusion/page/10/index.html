scriptprotect in CFMX7

I only recently found out about scriptprotect which is a cool new feature of ColdFusion MX7. Scriptprotect basically specifies whether to protect variables from cross-site scripting attacks, and can be enabled 2 ways. The first is a server setting in ColdFusion Administrator. Go to ‘Settings’ and check the “Enable global script protection” checkbox.
